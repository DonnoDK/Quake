Overview

This is the Cocoa version of the popular game Quake. It includes the software and OpenGL renderer versions of Quake and QuakeWorld.

System Requirements:

  • An Intel Mac.

  • MacOS 10.6 or later.

  • The shareware or retail version of Quake.

The port comes with the following files:

  • Quake: The software renderer version of Quake.

  • QuakeWorld: The software renderer version of the QuakeWorld client.

  • GLQuake: The OpenGL renderer version of Quake.

  • GLQuakeWorld: The OpenGL renderer version of the QuakeWorld client.

  • qwsv: The QuakeWorld server. A command line application.

  • qw: A folder required by the QuakeWorld server software.

  • Extras/cheapo: A proxy for QuakeWorld. A command line application.

  • Extras/vispatch: Required to apply water transparency patches. A command line application.

  • Extras/AppleScript Examples/: This folder contains several example AppleScripts for use with Quake.
